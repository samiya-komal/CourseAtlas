# 🌍 CourseAtlas – Global Course Finder Platform

### 🎯 Discover courses across 300+ universities worldwide



## 📌 Overview

**CourseAtlas** is a data-driven project that collects, cleans, and organizes course information from multiple university websites into a centralized format.

The goal is to simplify how students explore higher education opportunities by providing structured and accessible data.



## 🚀 Features

* 🌐 Automated web scraping of university websites
* 🔄 Handles multiple sources with different structures
* 🧹 Data cleaning and preprocessing
* 📊 Structured datasets in CSV format
* ⚙️ Master algorithm for running the pipeline



## 🛠️ Tech Stack

* Python
* BeautifulSoup
* Selenium
* Pandas
* Jupyter Notebook



## 📂 Project Structure

```
CourseAtlas/
│
├── notebooks/     # Scraping notebooks + MasterAlgorithm
├── CSV_FILES/     # Cleaned datasets
├── docs/          # Diagrams / screenshots
├── requirements.txt
└── README.md
```

---

## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Open Jupyter Notebook:

```
jupyter notebook
```

3. Run:

* Individual university notebooks
* OR `MasterAlgorithm.ipynb`

---

## 📊 Sample Output

The project generates structured datasets containing:

* Course Name
* Duration
* Fee Structure
* Admission Requirements

Example files are available in the `CSV_FILES/` folder.
